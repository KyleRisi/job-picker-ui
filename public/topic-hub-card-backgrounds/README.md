# Topic Hub Card Backgrounds

Drop your card backing images in this folder.

## Naming
Use the discovery term slug as the filename:

- `true-crime.jpg`
- `history.png`
- `incredible-people.webp`

## Supported file types
- `.avif`
- `.webp`
- `.png`
- `.jpg`
- `.jpeg`

The Topics hub cards automatically use `/topic-hub-card-backgrounds/<slug>.<ext>` when a matching file exists.
